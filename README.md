## Overview

LUXE Paris is a modern e-commerce store designed for luxury retail. The application provides an elegant and smooth user interface for browsing premium products, managing shopping carts, and completing purchases.

The store includes:
- Product display with full details
- Integrated shopping cart system
- Multi-step checkout process
- User account creation
- Search and filtering interface
- Responsive design for all devices

## Technologies

Built with modern cutting-edge technologies:

- **React 19** - Main user interface
- **TypeScript** - Strong and safe programming
- **Vite** - Fast development build tool
- **Tailwind CSS** - Modern CSS framework
- **Lucide React** - Beautiful and modern icons
- **Motion** - Smooth and interactive animations

## How to Run

### Prerequisites
- Node.js (version 18 or newer)
- npm or yarn

### Setup Steps

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Open application in browser:**
   ```
   http://localhost:3000
   ```

### Additional Commands

- **Build production version:**
   ```bash
   npm run build
   ```

- **Preview production version:**
   ```bash
   npm run preview
   ```

- **Code linting:**
   ```bash
   npm run lint
   ```
